# Codex Integration

This directory contains the Codex-ready export for `skill-accuracy`.

## Layout

- `skills/skill-accuracy/`

## Recommended install targets

- project-local vendor path such as `tools/skill-accuracy/codex/skills/`
- user-global Codex path `~/.codex/skills/`

## Quick install

Project-local:

```bash
bash scripts/install_codex.sh --project "$(pwd)"
```

User-global:

```bash
bash scripts/install_codex.sh --home
```
